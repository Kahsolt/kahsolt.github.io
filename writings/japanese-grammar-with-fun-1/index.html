日本語文法ノート(一)


这里是对王玉英《日语语法原来这么有趣》的笔记，第一章25个重要语法概念

基础概念

日语语法 = 单词 + 敬语 + 句型 + 语法

品词：

  • 名词
  • 连体词/冠词：こそあど言葉，修饰名词
  • 代名词/代词：人称代词、指示代词、疑问代词、关系代词、数量代词
  • 形容词/形容动词：-い/-だ
  • 动词：-u，做谓语是有时态(过现将)/语态(主被)/语体(简敬)变化
  • 副词：程度副词、陈述副词、情态副词
  • 助动词：判断、否定、过去/完成、推量、……
  • 助词：格助词/结构助词、句子连接关系、句末语气词
  • 接续词/连词：因果、并列、让步、条件、递进
  • 感叹词
  • *形容词/形容动词/动词/助动词是有活用的

敬语：

  • 尊敬语：いらっしゃる/なさる/お…になる/どなた
  • 谦逊语/自谦语：拙者/伺う/拝見する/差し上げる/ご返事する
  • 丁宁语/郑重语：お-/ご-/です/ます/ございます

句型：

  • 主语 + 场所にあります/います
  • 主语 + 数量代词 + 谓语(自动词)
  • 主语 + 场所で + 宾语を + 谓语(他动词)
  • 敬体否定式 + 疑问词(…ませんか)
  • 动词未然形 + なければなりません

基础词类 & 连接形

不同品词间连接用的结构助词：

  • 名の + 名
  • 形い/形动な + 名
  • 形て/形动で + 形/形动
  • 形く + 动/ない
  • 形动に + 动
  • 动て + 动
  • 某些自动词 + を，表示离开的场所
  • 某些自动词含有被动色彩：捕まる/振る
  • *用于列举的结构
    • …と…
    • …や…や…など
    • …やら…やら
    • …なり…なり
    • …とか…とか
    • …だの…だの
    • …たり…たりする

名词

名词：没有词尾，通过黏着成分です+ある来变形

敬/简 肯 否
现/将 Nです / Nだ Nではありません / Nじゃない
过 Nでした Nではありませんでした / Nじゃなかった

形容词/形容动词

形容词：以い为词尾，连体连用时变为く、过去式变为かっ

敬/简 肯 否
现/将 ADJい(です) / ADJい(だ) ADJくありません / ADJくない
过 ADJかった ADJくありませんでした / ADJくなかった

*特殊变形：いい

敬/简 肯 否
现/将 いい/よい よくありません/よくない
过 よかった よくありませんでした/よくなかった

形容动词：以だ为词尾，连体连用时变为な、时态变化同名词

敬/简 肯 否
现/将 ADJ(です) / ADJ(だ) ADJではありません / ADJではない / ADJじゃない
过 ADJでした ADJではありませんでした / ADJではなかった / ADJじゃなかった

*では = じゃ

动词

动词原形/字典形(U)：

  • 一段动词：-iru/-eru结尾，以る为词尾
    • *例外：要る/入る/帰る/蹴る/切る/知る/滑る/走る，这些是五段
  • 五段动词：其他-u结尾，没有词尾
  • か变/さ变动词：する/くる

动词丁宁形(I)/ます形：ます形 = 连用形 + ます

  • 一段动词：去掉词尾る加ます，ie. いる -> います
  • 五段动词：词末假名改为对应的い段假名，ie. ある -> あります, いく -> いきます
    • *特殊五段る -> ります：入る -> 入ります, 知る -> 知ります
  • か变/さ变动词
    • する -> します
    • くる -> きます
敬/简 肯 否
现/将 Vます/V Vません/Vない
过 Vました/Vた Vませんでした/Vなかった

*此处Vます即连用形，Vない是否定形，Vた是过去式

*特殊变形：有る(ある)

敬/简 肯 否
现/将 あります/ある ない
过 ありました/あった ありませんでした/なかった

动词进行式/て形：て形 = 连用形 + て/いて/いで/って/んで/して

  • 一段动词：去掉词尾る加て，ie. いる -> いて
  • 五段动词
    • く -> いて, ぐ -> いで (特殊：行く -> 行って)
    • う/つ/る -> って
    • ぬ/ぶ/む -> んで
    • す -> して
  • か变/さ变动词
    • する -> して
    • くる -> きて

动词过去式/た形：た形 = 连用形 + た/いた/いだ/った/んだ/した

  • 一段动词：去掉词尾る加た，ie. いる -> いた
  • 五段动词
    • く -> いた, ぐ -> いだ (特殊：行く -> 行った)
    • う/つ/る -> った
    • ぬ/ぶ/む -> んだ
    • す -> した
  • か变/さ变动词
    • する -> した
    • くる -> きた

动词否定形(A)/ない形：ない形 = 连用形 + ない

  • 一段动词：去掉词尾る加た，ie. いる -> いない
  • 五段动词：词末假名改为对应的あ段假名(但是う -> わ)，ie. ある -> あわない, いく -> いかない
  • か变/さ变动词
    • する -> しない
    • くる -> こない (注：音变为こ)
  • *过去式否定形即ない -> なかった

动词意志形(O)) = 连用形 + よう

  • 一段动词：去掉词尾る加よう，ie. みる -> みよう
  • 五段动词：词末假名改为对应的お段假名、再加う，ie. はなし -> はなそう, いく -> いこう
  • か变/さ变动词
    • する -> しよう
    • くる -> こよう
  • *常见应用：意志形 + と思う

动词命令形(E)) = 连用形 + -e/ろ/よ

  • 一段动词：去掉词尾る加ろ或よ，ie. みる -> みろ
  • 五段动词：词末假名改为对应的え段假名，ie. はなす -> はなせ, いく -> いけ
  • か变/さ变动词
    • する -> しろ/せよ
    • くる -> こい　(注意音变)

*总结：动词的元音色彩

  • A + ない: 否定形，”不……”
  • I + ます: 丁宁形，敬语
  • U: 普通形，简语
  • E (+ ろ/よ): 命令形，”请……吧”
  • O: 意志形，”想着……”

假定形 ければ/なら

  • 形容词：い -> ければ，ie. ない -> なければ, しろい -> しろければ
  • 名词/形容动词：词干加なら，ie. あめ -> あめなら, ひまだ -> ひまなら
  • 动词
    • 一段/五段：词末假名改为对应的え段假名、再加ば，ie. みえる -> みえれば, はなす -> はなせば, いく -> いけば
    • か变/さ变
      • する -> すれば, しない -> しなけえば
      • くる -> くれば, こない -> こなければ

连体形 (用言连体、作定语)

  • 形容词：即原形
  • 形容动词：だ -> な
  • 动词：即原形

かもしれない/かもしれません

だろう/でしょう
